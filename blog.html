Setup a remote git repository on your dreamhost server

Tuesday, July 9 2015

This blog will detail the technical things I manage to learn-- big and small. So, recently making the jump from windows to mac, one of the things I was excited to setup was a git repository for projects. I had my git on my local host all setup, my git linked to my github successfully, now I wanted to setup one to the actual server I am using for this very website. I pretty much follow the instructions provided by git here .

What we need to do is pretty simple.
  • Login to our remote dreamhost server via SSH
  • Initialize a git repo on our remote server
  • Switch back to our local host and tell our local git about the remote server.

Login to our remote dreamhost server via SSH

Dreamhost (and I think other server providing services?) provides a user for you, but you must change it to have shell capabilities-- this will allow us to login via SSH with this user to for command line access. This saves us a lot of headache from having to deal with setting up SSH keys! Dreamhost has done everything regarding that for us.

Login to your server via ssh from terminal

$ ssh dreamhostuser@radagast.dreamhost.com
dreamhostuser@radagast.dreamhost.com's password:
$ cd /home/dreamhostuser/myprojectdir
$ git init --bare
Initialized empty Git repository in /home/dreamhostuser/.git/
$ exit #exit connection, go back to local host..


We are creating an empty (bare) repository so that we can push our local stuff to. Note that this will add some stuff into your projectdir. Not sure what all of it is, something to do with git configuration..

Women Who Code NYC Meetup -- Lightning Talks

Tuesday, July 7 2015

Bienvenidos a Nueva York! I intend to keep most of my posts technical, but for my first one I want to write about the Women Who Code meetup I had the opportunity to attend yesterday.

I saw 10 lovely ladies speak. It was my first event like this with all women, and I loved it. Of course diversity of all kinds of is necessary for great collaboration, but there is something very relaxing about working with women, as a woman. One of the reasons I love programming so much is because of the lack of discrimination and BS that, comparitively, seems to accompany other professions. In programming I often feel the premise tends to be, "Can you make it work? Great." No one cares who you are, what you look like, where or even if you went to college, just prove you have the skills and enjoy making things work. Being surrounded by action-oriented women extends the attitude of minimizing BS. It inspires creation and acceptance --even encourgement-- of failure to dominate the culture of thinking. (Maurya Couvares, the founder of ScriptEd , even spoke about the importance of learning how to fail in the context of her company). Its just nice to be with your peeps.

Speakers discussed the technical-- js promises, ES6 features , overview of Swift, web accessibility -- and nontechnical alike.

One thing worth noting were 2 vibrant ladies who have recently started the company monarq-- a friendship app for women. Even more interestingly, they also started a women's hackathon which it think is fan-freaking-tastic. I definitely intend on going to in October.

Another women's oriented world-wide collaboritive event that was discussed was Womens Entrepretneurship Day .
